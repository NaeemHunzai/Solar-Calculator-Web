Engineering Notes

  • Corrected cold-weather Voc must remain below the inverter's maximum DC input voltage.
  • String operating voltage should remain within the inverter MPPT voltage range.
  • Operating current and short-circuit current must not exceed inverter limits.
  • The calculator uses the lowest ambient temperature to determine the worst-case open-circuit voltage.
  • Always verify the final design using the manufacturer's inverter and module datasheets.
